<details><summary>符号解释</summary>

表格内容

|符号|解释|
|:-----:|:----:|
|√|已经支持或已经适配的内容|
|!|已经支持或已经适配但未在游戏内实装可能过时的内容|
|×|暂未支持或暂未适配会在未来更新的内容|
|~|已经在制作中但未制作完成或未适配的内容|
|*|不完全支持或不完全适配的内容|
|/|不支持或不适配且未来没有计划更新的内容|

#### 主标语

|标语|解释|
|:-----:|:----:|
|驻场|角色作为出战角色在循环中占用较长时间进行输出，并且为队伍的核心输出角色|
|速切|角色作为队伍中主要输出但不占用较长的出战时间，但不以非出战状态输出作为主要输出手段|
|输出|角色在循环中不占用较长的出战时间，主要以非出战状态进行输出|
|辅助|角色主要在队伍中为自己或队伍中其他角色提供各种增益或状态效果|
|功能|角色在队伍中起到特殊功能或改变输出环境的作用|
|生存|角色在队伍中为其他角色承担伤害或为其他角色提供治疗或护盾|
|挂件|角色在队伍中没有主要作用，主要以凑天赋或凑元素共鸣为目的|
